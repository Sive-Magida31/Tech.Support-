Troubleshooting Guide

Follow these steps to diagnose and solve common technical issues.

1. Initial Assessment

What issue are you experiencing?

When did this problem first occur?

2. Environment Details

What device are you using? (e.g., Desktop, Laptop, Mobile)

What operating system is installed? (e.g., Windows, macOS, Linux, iOS, Android)

Which application or software are you using?

3. Connectivity Check

Are you currently connected to the internet?

If yes, can you access other websites or services?

4. Recent Changes

Have there been any recent changes to your system? (e.g., updates, new installations, hardware changes)

5. Error Identification

Are you receiving any error messages?

If yes, what does it say?

Can you reproduce the issue?

If yes, what steps lead to the problem occurring?

6. Troubleshooting Actions

Have you tried any troubleshooting steps? (e.g., restarting your device, reinstalling the application)

Have you checked for updates?

If yes, is your software and operating system up to date?

7. Desired Outcome

What is your desired outcome?

If yes, what resolution are you hoping for?

8. Additional Information

Is there anything else you think might be relevant?

If yes, any other details that could help diagnose the issue?
